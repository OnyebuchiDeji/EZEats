##  What Special Thing Was Done
+  Implementation of server-side pagination, server-side rendering, and asynchronous javascript (vanilla) in rendering of pages.
+  Efficient use of cookies, sessions, and client-side local storage for better user experience (UX).
+  Proper sanitization of product search queries for preserved security.
+  Use of cookies and client-side local storage of information, along with efficient building of query strings upon communication with databases to minimize
    number of requests to server side, and hence faster response on client-side.
+  Elaborate UI/UX for seamless user experience
